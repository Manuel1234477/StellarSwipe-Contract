# Deployment State

`deployments/testnet.json` is generated by the deployment scripts and consumed by the
nightly e2e test harness under `tests/e2e/`.

Expected fields:

```json
{
  "network": "testnet",
  "rpc_url": "https://soroban-testnet.stellar.org",
  "network_passphrase": "Test SDF Network ; September 2015",
  "contracts": {
    "governance": { "contract_id": "C..." },
    "signal_registry": { "contract_id": "C..." },
    "fee_collector": { "contract_id": "C..." },
    "user_portfolio": { "contract_id": "C..." },
    "oracle": { "contract_id": "C..." }
  },
  "assets": {
    "fee_token": { "address": "C..." }
  }
}
```

The e2e runner accepts either plain string IDs or the object form shown above.
